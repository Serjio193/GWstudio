# Bundled Third-Party License Texts

This directory contains license texts and notices for third-party components bundled with GW Studio release binaries.

Some dependency license files depend on the exact binary package used during build, especially Git for Windows / MSYS2, GNU Make, GNU Arm Embedded Toolchain, Python packages, and Windows runtime DLLs.

Before publishing a final release, verify the actual files bundled into `tools.zip` and replace placeholder files with upstream license/notice files from the exact packages used.
