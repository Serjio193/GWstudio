MinGW Runtime License Notice

GW Studio may bundle MinGW/MSYS2 runtime DLLs through Git for Windows, GNU Make, or other Windows command-line build tools.

The exact MinGW/GCC runtime license notices depend on the exact binary packages placed into tools.zip for the release build. Before redistributing a final binary package, verify the bundled files and include the matching upstream MinGW-w64 / GCC Runtime Library Exception / winpthreads license notices if those files are present.

Do not treat this README as a replacement for the exact upstream runtime license files.
